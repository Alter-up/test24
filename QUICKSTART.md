# ⚡ Быстрый старт

## 1. Установка (5 минут)

```bash
# Перейдите в папку проекта
cd x_automation_bot

# Запустите установку
chmod +x setup.sh
./setup.sh
```

## 2. Настройка Telegram бота (5 минут)

### Получение токена
1. Откройте [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте `/newbot`
3. Введите имя бота (например: `My X Bot`)
4. Введите username бота (например: `my_x_bot`)
5. Скопируйте токен (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Получение вашего ID
1. Откройте [@userinfobot](https://t.me/userinfobot)
2. Он покажет ваш ID (например: `123456789`)

### Редактирование .env
```bash
nano .env
```

Заполните:
```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
ALLOWED_USERS=123456789,987654321
```

## 3. Запуск

```bash
python main.py
```

Вы увидите:
```
Starting X Automation Bot
Database initialized
Starting Telegram Bot...
Starting API Server...
API will be available at: http://127.0.0.1:8000
```

## 4. Проверка работы

### Telegram бот
1. Найдите своего бота в Telegram по username
2. Отправьте `/start`
3. Должно появиться приветственное сообщение

### Web Dashboard
1. Откройте http://localhost:8000 в браузере
2. Должен загрузиться дашборд

### API Docs
1. Откройте http://localhost:8000/docs
2. Должна загрузиться Swagger документация

## 5. Добавление первого аккаунта

### В Telegram
1. Отправьте `/add_account`
2. Введите username X-аккаунта (без @)
3. Введите email от X-аккаунта
4. Введите пароль

### Проверка логина
```
/test_login @username
```

## 6. Добавление прокси (рекомендуется)

```
/add_proxy
```

Формат:
```
My Proxy|192.168.1.10:1080:user:pass
```

## 7. Первая автоматизация

### Лайк твита
```
/like
```
Затем отправьте URL твита.

### Подписка на пользователя
```
/follow
elonmusk
```

### Отложенный пост
```
/schedule
```
Следуйте инструкциям бота.

## 8. Настройка автоматизации

```
/automation
```

Выберите:
- **Lists** — автодействия из X-списка
- **Search** — автодействия по поиску
- **Autopost** — автопостинг по расписанию

## Полезные команды

| Команда | Описание |
|---------|----------|
| `/status` | Активные автоматизации |
| `/logs` | История действий |
| `/accounts` | Список аккаунтов |
| `/proxies` | Список прокси |
| `/help` | Полная справка |

## Остановка

Нажмите `Ctrl+C` в терминале.

## Перезапуск

```bash
python main.py
```

## Проблемы?

1. **Бот не отвечает** — проверьте токен в .env
2. **Ошибки логина** — проверьте логин/пароль, попробуйте без прокси
3. **Действия не работают** — проверьте логи в `x_automation.log`
4. **API недоступен** — проверьте, что порт 8000 свободен
